A web designer is an IT professional who is responsible for designing the layout, visual appearance and the usability of a website. A good web designer needs to have both creative graphic skills and technical skills. They need to be able to visualise how a site will look (the graphical design of the site - using HTML & CSS) and how it will function (conversion of a design into a working website - using JavaScript). The term Web Designer is often incorrectly interchanged with the term Web Developer and vice versa. A web developer is often more likely to be a software developer who works with programming languages to create a higher level of interactions on a website such as the integration with a database system. On the completion of this CodeFirst: Girls course, we can say we are web designers (beginners, anyway).

Cybersecurity analysts help prevent attacks through their expertise and knowledge of databases, networks, hardware, firewalls and encryption. They keep computer systems running smoothly, prevent the theft of financial and personal information, and block intruders from accessing and divulging proprietary data. They are essentially data bodyguards. If you're interested in the protection of data or solving cyber crimes and hacks, there are many free online courses to give you the introduction you need.

A cloud architect is an IT professional who is responsible for overseeing a company's cloud computing strategy, essentially where information is. This includes cloud adoption plans, cloud application design, and cloud management and monitoring. You've probably heard the phrase "it's in the cloud" and have thought "what is the cloud?". To be honest, it's a question many don't know the answer to, but simultaneously don't ask. 'The Cloud' is generally used to describe data centers available to many users over the Internet. With less jargon: it is a way to store data over the internet as opposed to your computer hard drive.
